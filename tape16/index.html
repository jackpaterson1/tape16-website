Tape Workflow

A DAW workflow that feels like a reel to reel tape machine.

If you are searching for a digital tape machine, virtual tape machine, or reel to reel style 16-track recorder for DAW writing, TAPE 16 is built to keep you recording instead of over-editing.

  • Fixed 16-track layout for faster arrangement decisions
  • NO UNDO tape workflow: commit the take or re-record it
  • Transport-led control inspired by real tape workflows
  • A focused DAW experience with modern plugin support

Remote Control

Remote control from any device.

Open the new phone remote web app in a browser to arm tracks, drop markers, toggle rec safe, and run transport controls from any device.

Workflow

Built for flow, not endless setup.

TAPE 16 keeps you focused on recording decisions. Arm a track, hit transport, capture the take, and keep moving.

  • Hardware-style REW, FF, STOP, PLAY, and REC controls
  • Per-track arm, solo, and mute with clear status states
  • Tape character and calibration controls where they matter
  • Realtime and offline bounce options

Minimum Requirements

macOS

  • macOS 12 (Monterey) or later
  • Apple Silicon M1 or newer
  • 16 GB RAM minimum (recommended for larger sessions)
  • 500 MB free disk space minimum
  • CoreAudio-compatible audio device (built-in or external)
  • Internet connection for license activation and updates

Windows

  • Windows 10 or Windows 11 (64-bit)
  • Intel Core i5 (12th gen+) or AMD Ryzen 5 5600X+
  • 16 GB RAM minimum (recommended for larger sessions)
  • 500 MB free disk space minimum
  • WASAPI or ASIO-compatible audio device (built-in or external)
  • Internet connection for license activation and updates
