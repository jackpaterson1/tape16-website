Release Notes

TAPE 16 0.9.13

Published February 26, 2026

UI performance and refresh consistency

Set default UI refresh to a steady 60 FPS across idle/play/record/rewind/fast-forward, and added Calibration FPS options: 24 / 30 / 60.

FX and Send FX plugin search workflow

Added inline TYPE TO SEARCH rows directly inside FX slot menus (Track/Bus/Master/Send) with live top results, and improved FX menu interaction stability.

Plugin persistence reliability

Plugin insert/remove changes now persist immediately, and UAD restore persistence was fixed by making protected VST3 startup skip opt-in only.

MIDI and instrument stability

Fixed held-note blip behavior by keeping instrument tracks rendering between MIDI events, plus added a MIDI input safety fallback if exclusions leave zero inputs open.
